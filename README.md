**Wanpachi bot**
> 動作要件
```
Node v16.6以上
npm v7 以上
Discord.js v13
```
> 必要パッケージ
```
Discord.js v13
```
> インストして実行
- node_moduleは同封しているのでダウンロードしてすぐ実行できます
- コンソールで
```
node index.js
```
と実行して
```
○○.js is loaded!
bot is ready!
```
と表示されたら成功。
